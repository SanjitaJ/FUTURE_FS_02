# FUTURE_FS_02

LeadFlow - Client Lead Management System (Mini CRM)
Overview

LeadFlow is a modern Client Lead Management System (Mini CRM) developed as part of the Full Stack Web Development Internship at Future Interns.

The application helps businesses, agencies, and freelancers efficiently manage client leads generated through website contact forms. It provides a centralized dashboard for tracking leads, updating statuses, managing follow-up notes, and monitoring conversions.
